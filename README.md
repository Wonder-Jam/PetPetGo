# 在模拟器中测试
因为是元服务，所以安装之后直接在桌面还找不到嘞，但是可以在模拟其中Ctrl+鼠标往中间划   
（这相当于在模拟两根手指往中间划），就可以找到元服务了

# 项目简介
- EntryAbility.ts:程序入口
- pages:程序页面
- EntryformAbility.ts:卡片管理类
- widget/pages:小卡片页面
- resources:各种静态资源
